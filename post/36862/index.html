# 跨境通 VISA / 万事达借记卡介绍与网上支付体验

本文篇幅较长，建议使用左侧导航菜单选择章节阅览

# 介绍

  • 长城跨境通国际借记卡是中国银行 四大国有银行之一 发行的 VISA / MasterCard 多币种外标借记卡。不设申请门槛， 支持网上申请和线下申请 ，适合学生等不具备申请信用卡条件的人员。
优点
  1. 真正的国际借记卡
  不上信报，不是某些银行以借记卡之名发行的零额度信用卡。

  2. 不设申请门槛，支持网上申请和线下申请
  只需本人有效身份证件，无需收入证明等财力证明材料。
  中行上海分行微信公众号提供网上申请通道。

  3. 支持 19 种外币，消费不收取任何手续费
  消费或 ATM 取现支持直接使用美元、欧元、日元、港币等 19 种外币，无需货币转换。
  免收货币转换费、跨境手续费等消费手续费。每月免收首笔境外 ATM 取现和余额查询手续费。

  4. EMV 芯片卡，支持非接触功能
  国内首张搭载 EMV 芯片标准的国际借记卡，属于磁条芯片复合卡。
  支持 Visa payWave 和 MasterCard Contactless 非接触式支付功能。

  5. 各种卡权益和银行优惠活动
  可以享受卡组织提供的卡权益和参与银行举办的各种优惠活动。
缺点
  1. 存在年费
  每年收取最低 RMB ¥10，最高 RMB ¥50。免收首年年费。(听说新政策借记卡取>年费，不知真假)

  2. 不设人民币账户
  需要手动购汇，将人民币兑换成外币。

  3. 使用上存在诸多不便
  使用跨境通卡的时候经常需要避开许多大大小小的坑。相比信用卡，用卡较为不方。

  4. 网点经常缺少卡板
  导致线下办卡以及申请不支持网申的卡片十分困难。
常见误区 本章收集了各大社区、论坛上对跨境通卡的一些误区。
  1. 跨境通卡是零额度信用卡 跨境通卡是真正的 VISA / MasterCard 国际借记卡。除了卡组织不同，跨境通卡与中行发行的银联借记卡本质上是一样的。 可能把它当成了中行早期发行的另一“国际借记卡” - 长城环球旅行卡。该卡于 2016 年停发，虽然对外宣传是借记卡，但实际上是一张零额度信用卡。

  2. 跨境通卡是单币种卡 / 是美元卡 / 只有美元账户 跨境通卡是多币种卡，支持美元、欧元、日元、港币等 19 种外币。

  3. 跨境通卡消费冻结 105% / 120% / 150% 跨境通卡美元消费冻结 100%、非美元消费冻结 102%。冻结比例与交易币种有关。 发卡早期（2017～2018年左右）非美元消费的冻结比例是 105%，后来调整至目前的 102%。

  4. 跨境通年费是 10 美金 / 20 美金 / 50 美金 跨境通卡不同卡等级的年费分别为10 人民币、20 人民币、50 人民币。 年费的收取方式是扣取等值的美元外钞。

  5. 跨境通卡消费存在手续费 除年费和 ATM 取现费之外，跨境通卡消费不收取任何额外费用，例如货币转换费、跨境手续费等等。 非美元消费额外冻结的 2% 资金不是手续费，入账后返还至原卡。

  6. 跨境通卡是国内唯一一种可以申请到的外标借记卡 中信银行于 2019 年 7 月新发行了一款 MasterCard 外标借记卡。 此外，北京银行、工商银行等银行都有发行外标借记卡（申请难度、用卡体验不同）。

# 申卡指南

# 申卡前的准备工作

  1. 年满 16 周岁
    按照中行规定，未满 16 周岁人士必须在监管人陪同下在网点申卡。未满 16 周岁不支持网申。

  2. 持有中行发行的银联借记卡，满足：

持卡人为本人 本人名下
属于 I 类账户 一类卡

  1. 已开通 中行手机银行
    用于购汇、转账和查看交易明细。

  2. 持有的 中行实体借记卡数量必须小于 4 张
    中行系统限制，每人最多只能持有 4 张实体借记卡 包括银联借记卡和 VISA / MasterCard 借记卡 。

# 申卡后的准备工作

  1. 下载中行手机银行， 将卡片关联到手机银行上

  2. 通过手机银行， 向卡片转账任意金额的外汇 / 外钞
    若借记卡开卡后 6 个月内没有发生任何动账交易，停止非柜面业务。

  3. 用卡前，通过手机银行 开通卡片的网上支付功能

  4. 若新卡经常遇到付款失败等异常情况，请停止用卡 3 ～ 5 天。
    仅限部分开卡不足 3 天的新卡。具体原因不明。

  5. 如果长时间不用卡 一年以上 ，卡片余额必须不为零
    否则被中行视为 双零卡 ，按照 相关规定 进行定期销户。

  6. 保留足够的美元外钞余额，用于缴纳年费

手机银行开通网上支付功能
  需要开启下列功能：

  1. 卡组织在线支付
  2. 卡组织代收
  3. 卡组织订购业务
  4. 小额/凭签名免密

  手机银行选项位置：

  1. 卡组织在线支付、卡组织代收、卡组织订购业务**：账户管理 -> 跨境通卡 -> 支付设置 -> 跨行支付
  2. 小额/凭签名免密**：账户管理 -> 跨境通卡 -> 限额设置 -> 小额/凭签名免密(POS消费)

  找到开关后，按照客户端上的指引开启功能即可。限额单位是人民币

# 申卡流程

  1. 网上填写资料
    按照网申链接上的指引操作即可。需要提交的资料有身份证正反面、头像照、个人信息、邮寄地址必须 精确到门牌号 ，不然审批不通过。
  2. 进度查询
    如果 当前状态 变为 制卡中 ，说明申请成功。

# 注意事项

快递单号 存在 BUG 。经常出现收到卡后才会显示快递单号的情况。

如果 审核未通过 ，可能是没有填写详细的邮寄地址。

如果 开卡失败 ，原因不明，可能是上海分行系统出问题。需要重新申请。

  1. 激活卡片
    一般过两到四周后，可以收到上海分行邮寄的卡片挂号信。

按照挂号信的指示，前往全国任意中行网点，在智能柜台上自助激活即可。

# 网点线下

  1. 优点： 速度快 。若不考虑取号排队时间，一般可以在 30 分钟内办到跨境通卡。
    因此这是首选的申请跨境通卡方式。
  2. 缺点：
  1. 缺少卡板 ：由于跨境通卡是相对冷门的卡种，许多网点 特别是位于小城市、城镇的小网点 没有提供跨境通卡卡板。即使是一两线大城市的网点，也往往只提供数量有限的卡板。
  2. 柜员操作不熟练 ：原因同上。虽然这是小概率事件，如果柜员操作出错，后果就是 无法正常用卡、只能销掉重新申请 。

# 申卡流程

  1. 寻找开卡网点
    拨打网点座机或者直接去现场询问。建议先找离自己最近的网点，然后再找当地的大型网点 如市分行、省分行 。
    大堂经理 站在入口附近协助客户办理业务的这位 一般不知道卡板的库存情况，建议让柜员 隔在玻璃窗后面办理业务的这位 去网点的卡库查找是否有可用的跨境通卡卡板。
    与网点人员交流时，他们可能不熟悉 跨境通卡 这个词，建议使用 VISA / MasterCard 借记卡 、 EMV 借记卡 或者直接把卡面照片给他们看。

  2. 柜台办卡

# 购汇、交易提醒

# 购汇

  1. 详细教程：中国银行手机银行购汇与转账教程

  2. 跨境通卡 不支持人民币账户 。需要在手机银行上购汇 人民币兑换为外币 ，然后将购得的外币转账到卡片中。

# 注意事项

若无特殊需求，建议 购买外汇为主
外钞一般用于 ATM 取现和缴纳年费 。除了少数币种，外钞的价格一般比外汇贵。

（境内转账）外币只能转账给本人账户和直系亲属账户
本人的非中行账户、直系亲属的中行或非中行账户之间的外币转账需要到柜台上办理。
跨境转账不受限制。

# 避坑指南

# 交易方式

  1. 与外标信用卡相同，跨境通卡采取 先预授权、后入账扣款 的交易方式。即消费时不会立即扣款，而是先预授权 冻结 特定比例的资金，等待银行入账后再解冻资金扣款。入账时间一般是两至三天。

中行在 长城借记卡章程 第十一条给出的解释如下：

  1. 通过 VISA、MasterCard 等国际银行卡组织网络进行的交易属双信息交易，即卡片发 生交易时，先在持卡人的账户中冻结交易金额，当国际银行卡组织与中国银行进行 结算时再从持卡人账户中完成扣款。
  • 举个例子：使用美元汇户支付 $600 美元：
原状消费 $600入账后
美元汇户余额USD $700USD $700 $100 可用USD $100

注： 在手机银行上，如果账户存在预授权 冻结 资金，账户余额的旁边会标有 XXX 可用 。

  • 账户余额减去括号中的数字即是预授权金额。

# 有关消费明细和通知的坑

  1. 手机银行无法查看未入账交易
    手机银行只能显示所有未入账交易总和的预授权金额，既不能单独显示每一笔未入账交易，也不能查询商户名、消费金额和消费时间等交易信息。
    举个例子： 如果在某天连续刷了 $100、$200 和 $300 三笔交易，当天手机银行只能显示美元账户存在 $600 的预授权金额，除此之外没有更多信息。
    解决方法： 人工柜台使用 交易代码 037671 可以查 > 询到未入账交易。

  2. 微信交易提醒存在延迟
    微信交易提醒只推送入账交易通知、不推送预授权交易通知，即发生消费预授权时不发送通知，两至三天后入账时才发送通知，存在时间延迟。
    解决方法： 开通付费的短信交易提醒 每月 2 元 。

# 账户余额问题

  1. 绑卡或者日常消费时， 美元账户余额须不少于 2 美元
    若首次付款或者绑卡后长时间没有发生付款，为了验证卡片状态是否正常，网站一般先预授权 冻结 一笔小额资金 过一段时间后退还 ，预授权成功后才进行付款。
    预授权金额一般为 1 美元 ，可能存在以下情况。一般来说，2 美元足够应付大部分场合。
  • 超过 1 美元 ：如日区 Google Play 是 204 日元 合1.98美元 。
  • 当地货币 ：如日本亚马逊是 1 日元，德国 PayPal 是 1 欧元。
  1. 使用非美元账户消费，账户余额须不少于消费金额 × 102%
  • 跨境通卡预授权金额的比例与交易币种有关：
交易币种冻结比例备注
美元100%使用美元账户 支付美元或非美元
非美元102%使用非美元账户支付对应币种 （多冻结的 2% 入账后返还）
  • 举个例子：若使用英镑汇户支付 £440 英镑，账户余额必须不少于 £448.8：
原状消费 £440入账后
英镑汇户余额GBP £448.8GBP £448.8 £0 可用GBP £8.8

# 账户扣款顺序

  1. 跨境通卡支持以下 19 种外币。每种外币分为外汇和外钞，故最多可以拥有 38 个外币账户。
  • 1. 美元、欧元、日元、港币、英镑、加拿大元、澳大利亚元
  • 2. 瑞士克朗、新加坡元、澳门元、韩国元、丹麦克朗、挪威克朗
  • 3. 新西兰元、菲律宾比索、俄罗斯卢布、瑞典克朗、泰国铢、南非兰特

  1. 跨境通卡外币账户的扣款顺序为： 交易币种钞户 > 交易币种汇户 > 美元钞户 > 美元汇户

  2. 即先交易币种账户后美元账户、先钞户后汇户。若优先级高账户的余额不足，自动使用优先级较低账户，依此类推。

  3. 中行官方的说明如下 注： 长城借记卡章程 第十条。

    VISA、MasterCard 多币卡账户如有原始交易货币的存款且余额充足，直接在该账户扣款，否则全额轮循美元账户扣款。外币账户的扣款顺序均为先钞户后汇户。

# 注意

  1. 不支持人民币账户
  2. 非美元的外币账户只能支付对应币种
    例如：英镑账户只能支付英镑，不能支付美元、港元等其他币种。
  3. 19 种外币以外的币种只能使用美元账户支付
    例如：新台币不属于 19 种外币之一。即使跨境通卡存有新台币外币，也只能从美元账户扣款。
  4. 美元账户支付非美元，使用 VISA / MasterCard 卡组织汇率
  • 官方提供的汇率计算器： VISA 、MasterCard
  • 第三方汇率计算器： VISA 、 MasterCard
  1. 不支持多账户支付
    只能使用一个外币账户全额支付，不能从多个账户扣款。

# 缴纳年费

  1. 年费收取方式：从美元钞户扣取等值的美元外钞
    注意，只能从美元钞户扣费，不是美元汇户。
  2. 年费收取时间：每年的开卡日，从开卡后第二年开始收取。
    开卡日可以在手机银行上查到 卡片信息->开户时间

# 有关缴纳年费的坑

  1. 没有任何通知
    无论扣取年费成功与否，中行 不会发送 向用户任何形式的通知或提醒 包括微信和短信交易提醒 。
    如果出现 批量收费 的交易记录，只能说明中行扣除了年费，不一定缴纳年费成功。如果美元钞户余额小于年费，中行仍会将余额全部扣除。
  2. 手机银行不能自助缴纳年费
    解决方法： 存入足量的美元外钞，等待下一次扣款。若当年的年费扣取失败或未缴齐，中行每个月会尝试再次扣款，直到收齐年费。

# 3D 验证服务

  • 3D验证服务 （Verified by Visa、MasterCard SecureCode) 是 VISA / MasterCard 卡组织验证持卡人的方式，常见于日本、香港等亚洲地区网站。若网站开启 3D 验证，付款前需要转跳到中行开设的 3D 验证网页，输入银行预留手机号和短信验证码，验证持卡人身份。

# 有关 3D 验证服务的坑

  1. 只支持短信验证
    中行信用卡 3D 验证可以支持短信和动态口令验证。
    如果手机信号不好或者家住境外，可能不方便接受到验证码短信。
  2. 若 3D 验证连续失败次数过多，3D 验证锁定且无法解锁
    注： 这部分内容博主有待进一步考察

# 帐单地址

  • 跨境通卡 不检验帐单地址 。跨境通卡与国内外标信用卡一样，不支持 AVS 系统验证。

# 人工柜台（交易代码 037671）

  • 这种方法可以查询到中行系统的内部交易记录，支持查询未入账交易。
交易代码交易名称
037671 （部分地区 037670）双信息授权记录
  1. 查询方法
    带上卡片和身份证去网点柜台，让柜员使用上列交易代码查询卡片的交易记录，指定想要查询的时间范围，打印查询结果即可。
  2. 判断交易状态
    通过清算日期和剩余圈存额可以判断交易的状态。
交易状态清算日期剩余圈存额
未入账空白不为 0
已入账不为空白0

其中 清算日期 指入账时间， 剩余圈存额 指处于冻结状态的资金金额。
如果 清算日期 为空白且 剩余圈存额 为 0，这是网站 验卡记录，可以忽略。
